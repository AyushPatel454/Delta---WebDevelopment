Grid-Template

They define the lines & tracking sizing.

grid-template-rows: 60px 60px 60px 60px

total number of Rows = 4 (each size = 60px)

grid-template-column: 100px 100px 100px

total number of Column = 3 (each size = 60px)

Another Method of Grid-Template

Repeat is used to divide all available space.

grid-template-row: repeat(count, 1fr)

grid-template-row: repeat(4, 1fr) -> 1fr 1fr 1fr 1fr

4 fraction equal number of rows bani jase.

grid-template-column: repeat(count, 1fr)

grid-template-row: repeat(3, 1fr) -> 1fr 1fr 1fr

3 fraction equal number of coloumn bani jase.

Common Properties.

  1. Horizontal

    • justify-items: start/center/end(container)
    • justify-self: start/center/end(item)

  2. Vertical

    • align-items: start/center/end(container)
    • align-self: start/center/end(item)

  3. Common for Horizontal & Vertical

    • place-items: start/center/end(container)
      • justify-items & align-items have same property.
    • place-self: start/center/end(item)
      • justify-self & align-self have same property.
