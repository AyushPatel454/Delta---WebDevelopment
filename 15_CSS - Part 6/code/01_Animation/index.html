Animation in CSS

To animate CSS elements.

Syntax:

@keyframe myname {
from {font-size: 20px;}
to {font-size: 40px}
}

Short-Hand of Animations.

animation: myName animation_duration animation_timing_function animation_delay animation_iteraion_count animation_direction

Example: animation: myName 4s ease-in 1s infinite normal
