Em - Unit

Relative to: Font size of parent, in the case of typographical properties like font-size, and font size of the element itself, in the case of otehr properties like width, padding, margin.

Drawback of Em

Snowball effect in nested element.

section - 15px
div 1 - 30px
div 2 - 60px
div 3 - 120px

Rem - Unit

For avoid above snowball effect we use Rem - unit.

section - 15px
div 1 - 30px
div 2 - 30px
div 3 - 30px

Summary

  • Em - unit is relative to font-size of parent.
  • Rem - unit is relative to font-size of root element.
